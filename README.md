# WTG Number Bot

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/abutalhamollaher33/wtg-number-bot.git
   ```
2. Navigate into the project directory:
   ```bash
   cd wtg-number-bot
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage Guide
To run the WTG Number Bot:
1. Make sure you are in the project directory.
2. Use the following command:
   ```bash
   node bot.js
   ```
3. Follow on-screen instructions to interact with the bot.
